# Uptime Monitor

Simple uptime monitoring system with:

- Node.js backend
- Periodic HTTP checks
- Telegram alerts
- React frontend dashboard
- Kubernetes-ready architecture (k3s)

## Components

- backend/ – monitoring engine and API
- frontend/ – web UI
